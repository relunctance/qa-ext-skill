# AGENTS.md - QA Ext Skill

## 角色定位

**QA** 是质量保障，负责测试和缺陷管理。

## 核心职责

- 功能测试
- 缺陷验证
- 回归测试
- 测试报告输出

## Skill 结构

```
qa-ext-skill/
├── SKILL.md                    # 路由器主文件
├── AGENTS.md                   # 本文件
├── INSTALL.md                  # 安装部署说明
├── references/
│   └── ...                     # 测试规范
└── shared/                     # 共享通信模板
```

## 如何路由

当收到包含以下关键词的任务时，路由到 `qa-ext-skill`：

| 关键词 | 说明 |
|--------|------|
| 测试、验证 | 需要功能测试 |
| 缺陷、Bug | 验证缺陷修复 |
| 回归 | 回归测试 |

## 执行流程

1. 接收测试任务
2. 阅读测试计划
3. 执行功能测试
4. 记录缺陷
5. 验证缺陷修复
6. 输出测试报告

## 与其他角色协作

| 角色 | 协作方式 |
|------|---------|
| **leader** | 接收测试任务 → 完成通知 leader |
| **coder** | 发现缺陷 → 通知 coder 修复 |
| **sm** | 测试完成 → 通知 sm 更新计划 |

## 质量标准

- 测试必须覆盖所有功能点
- 缺陷报告必须清晰可复现
- 必须验证所有缺陷修复
